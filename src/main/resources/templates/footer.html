이문관

- 담당자 이름 : 이문관

- 담당 페이지 : 회원정보 수정, 비밀번호 수정, 반려견 조회/등록/수정/삭제, 내 게시물 보기

- 상세 기능 :
1. 회원정보 수정 : 로그인한 회원의 정보를 수정할 수 있는 페이지. 해당 페이지에 접속하면 회원의 정보가 보여짐. 원하는 부분만 수정 가능.
2. 비밀번호 수정 : 사용자한테 기존 비밀번호를 입력 받은 후 DB의 비밀번호와 일치하는지 확인. 일치하면 수정 진행, 다르면 안내 문구 표시.
3. 반려견 조회 : 로그인한 회원이 등록한 반려견을 조회할 수 있는 페이지
4. 반려견 등록/수정/삭제 : 반려견을 등록 및 수정,삭제 할 수 있는 페이지
5. 내 게시물 보기 : 로그인한 회원의 게시물을 조회 할 수 있는 페이지

이상원

- 담당자 이름 : 이상원

- 담당 페이지 : 회원가입 / 로그인 / 뉴스페이지 크롤링 / 헤더

- 상세 기능 :
1. 로그인(일반) : 스프링 시큐리티와 DB를 연동하여 로그인 처리 및 세션 구현
2. 로그인(카카오) : 카카오 로그인 연동 및 DB와 연동하여 자동 로그인 구현
3. 회원가입(일반): 스프링 시큐리티와 DB 연동된 회원가입 구현
4. 회원가입(카카오) : 카카오 로그인시 등록된 이메일이 아닌 경우 회원가입페이지로 연동 비밀번호 수정불가
5. 헤더 : 로그인시 인증여부를 확인하여 메뉴가 숨김/보임을 타임리프로 구현
6. 뉴스페이지 : Jsoup을 사용한 웹크롤링으로 동물관련 뉴스 전달/페이징처리

이용제

- 담당자 이름 : 이용제

- 담당 페이지 : 자유게시판, 질문게시판, 모임게시판

- 상세 기능 :
1. 자유/질문 게시판 : 검색 및 조회(2가지 키워드), 게시글과 댓글 작성, 수정, 삭제
2. 모임 게시판 : 검색 및 조회(3가지 키워드), 게시글과 댓글 작성, 수정, 삭제
3. 사이트 배포
